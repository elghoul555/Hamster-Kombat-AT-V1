How to Upload a Project to GitHub

Step 1: Create a GitHub Account

If you don't already have a GitHub account, you need to create one.

  1. Go to GitHub.
  2. Click on Sign up and follow the instructions to create an account.

Step 2: Create a New Repository

After creating your account, you need to create a new repository for your project.

  1. Log in to your GitHub account.
  2. Click on the + button in the top right corner and select New repository.
  3. Fill in the details:
    • Repository name: Hamster-Kombat-AT-V1
    • Description: A simple application for automatic mining with a hamster character.
    • Select Public.
    • Check the box Add a README file.
  4. Click on Create repository.

Step 3: Install Git on Your Computer

You need to have Git installed on your computer to upload your project to GitHub.

  1. Download and install Git from git-scm.com.
  2. Follow the installation instructions for your operating system.

Step 4: Configure Git

Set up your Git username and email address. Open your terminal (Command Prompt, PowerShell, Terminal on Mac, etc.) and run the following commands:

git config --global user.name "your-github-username"
git config --global user.email "your-email@example.com"

Step 5: Initialize a Local Repository

Navigate to your project directory and initialize a new Git repository:

cd path/to/your/project
git init

Step 6: Add Project Files to the Repository

Add all your project files to the Git repository:

git add .

Commit the files to the repository:

git commit -m "Initial commit"

Step 7: Link the Local Repository to GitHub

Add the remote repository (the one you created on GitHub) as the origin:

git remote add origin https://github.com/your-github-username/Hamster-Kombat-AT-V1.git

Step 8: Push the Project to GitHub

Push your local repository to GitHub:

git push -u origin master

Project Structure

Your project should have the following structure:

Hamster-Kombat-AT-V1/
├── hamster_dance.gif
├── ham.py
├── README.md
└── requirements.txt

Step 9: Create a README.md File

Create a README.md file in your project directory with the following content:

# Hamster Kombat AT V.1

![hamster dance](hamster_dance.gif)

## Description
A simple application for automatic mining with a hamster character.

## Features
- Automatic mining with double-click every 150 ms.
- Selection of energy and boost levels.
- Automatic energy recharge.
- Multilingual support.
- Global hotkeys to start (F2) and stop (F3) mining.

## Requirements
- Python 3.x
- Libraries listed in requirements.txt

## Installation
1. Clone the repository:
   
git clone https://github.com/your-github-username/Hamster-Kombat-AT-V1.git

2. Navigate to the project directory:
   
cd Hamster-Kombat-AT-V1

3. Install the dependencies:
   
pip install -r requirements.txt


## Usage
Run the script:

python ham.py


## Contributing
Contributions are welcome. Please open an issue or submit a pull request to contribute.

## License
[MIT](https://choosealicense.com/licenses/mit/)

Step 10: Create a requirements.txt File

Create a requirements.txt file in your project directory with the following content:

pyautogui
pynput
pyperclip
pillow

Step 11: Update the Repository on GitHub

Add and commit the new files:

git add README.md requirements.txt
git commit -m "Add README and requirements"
git push

All Done!

Your project "Hamster Kombat AT V.1" is now uploaded to GitHub. You can share the repository link so others can view and contribute to your project.
